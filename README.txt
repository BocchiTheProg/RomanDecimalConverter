Програма для перетворення римської цифри в ціле число та навпаки.
Для введення вхідного значення використовується консоль.
При введенні римських цифр відбудеться конвертація до цілого числа, а при введенні цілого - відбудеться обернена конвертація.
Examples:
ruby Main.rb 1990
ruby Main.rb MMVIII
P.S. Римські числа записуються, виражаючи кожну цифру окремо.
